# Задание №2
## Вариант №7

Дан массив пар, содержащих начальный и конечный пункты пребывания. Вернуть итоговый конечный пункт самой длинной цепочки следования.

## Пример 1
### input:
  ["Новосибирск","Дубай"],
  ["Новосибирск","Искитим"],
  ["Искитим","Линево"]
### output:
Линево

## Пример 2
### input:
  ["Лондон","Стамбул"],
  ["Новосибирск","Лондон"],
  ["Стамбул","Пекин"]
### output:
Пекин

## Пример 3
### input:
  ["Барнаул","Новосибирск"]

Новосибирск
